# Basic Python Programs

This project contains two simple Python programs.

---

## 1. Math Operations (`math.py`)
This program:
- Takes two numbers from the user
- Performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays the results of each operation

### How to Run:
```bash
python.exe .\math.py
```

## Personalized Greeting (`message.py`)

This program:
- Takes the user's first name and last name as input
- Joins them together to form a full name
- Prints a personalized greeting message using the full name

### How to Run:
```bash
python.exe .\message.py
```
